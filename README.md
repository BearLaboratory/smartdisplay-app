# 1. 前言

为了方便我自己使用墨水屏显示终端，我不得不去写一个手机APP辅助自己。同时为了能加快开发最终选用uniapp开发框架进行APP的编写。其实uniapp你可以理解成安卓的webview就行了，通过jsbrage给页面调用硬件能力。

# 2.如何使用

具体使用请参考视频。请注意：在使用前请开启手机蓝牙，有些设备需要开启定位功能。

# 3.如何开发

要想进行开发，首先下载hbuilderX，然后导入项目，直接运行即可
